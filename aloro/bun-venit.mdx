<Info>
  **Aloro** este o platformă de agenți vocali AI care **sună și răspund la apeluri** în locul tău:
  colectează informații, urmează scenarii, salvează note și declanșează acțiuni - automat, la scară.
</Info>

## Ce poți face cu Aloro

<CardGroup cols={2}>
  <Card title="Construiești agenți vocali" icon="bot">
    Definești comportamentul prin prompturi, voce și reguli.
  </Card>
  <Card title="Rulezi apeluri" icon="phone">
    Inbound (primești) și outbound (suni), individual sau în masă (campanii).
  </Card>
  <Card title="Extragi date din apeluri" icon="database">
    Reguli AI care citesc transcriptul și salvează informații structurate.
  </Card>
  <Card title="Integrezi cu sistemele tale" icon="code">
    API și webhooks pentru a conecta platforma la software-ul tău.
  </Card>
</CardGroup>
